## CSS 작성 시 주의사항 및 기본순서
* css로 디자인 적용 전 html 태그의 고유디자인이 활성화 되어 있으면 안된다.
* (위)가 활성화 되어 있다면 css 명령어를 통해 html 디자인을 초기화 해야 한다.
* 초기화(reset.css)연결 후, color, font-family 정보 제작하는 디자인 특징에 맞게 재체크하기
* 초기화(reset.css)연결 시, 기존 디자인하는 다른 css 파일보다 반드시 먼저 연결하기(순서상 가장 위로)
* 디자인 css 진행 시, 디자인하려는 html 파일의 body 태그의 첫번째 자식 태그부터 선택자로 만들고 순서대로 진행하기
* 특정 대상에 `width, height` 크기 관련 속성 작성 시, 영역의 크기와 범위, 위치를 구분하기 위해 `background-color`를 설정하기(실제 디자인 배경색이랑 연관없이 구분 위한 테스트용으로 주로 영문명 사용)
